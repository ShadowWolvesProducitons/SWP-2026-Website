UPLOAD INSTRUCTIONS FOR VENTRAIP / CPANEL:
====================================================================

1. LOG INTO YOUR VENTRAIP CPANEL
   - Go to your VentraIP control panel
   - Navigate to File Manager

2. NAVIGATE TO YOUR PUBLIC DIRECTORY
   - Find and open: public_html (or www or httpdocs)
   - This is where your website files go

3. UPLOAD ALL FILES
   Option A - Upload ZIP:
   - Upload shadowwolves-website.zip
   - Extract it in public_html
   - Delete the ZIP file after extraction
   
   Option B - Upload Files Directly:
   - Select ALL files from the build folder
   - Upload: index.html, asset-manifest.json, and static folder
   - Make sure the "static" folder structure is preserved

4. SET PERMISSIONS (if needed)
   - Files: 644
   - Folders: 755

5. VERIFY YOUR WEBSITE
   - Visit: www.shadowwolvesproductions.com.au
   - Test all pages and navigation
   - Test the contact form (it will open your email client)
